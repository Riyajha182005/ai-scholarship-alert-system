# 🎓 AI Scholarship Alert System

## 🚀 Overview

This project is an AI-powered automation system that fetches real-time scholarship opportunities and shares them in a Telegram channel.

It is designed to help students especially women discover opportunities easily without manual searching.

---

## ⚙️ Features

* 📡 Fetches latest scholarships using RSS feeds
* 🤖 Uses AI to clean and format raw data
* 📢 Automatically sends alerts to Telegram
* 🔁 Fully automated workflow using n8n

---

## 🛠 Tech Stack

* **n8n** – Workflow automation
* **LLM (AI Model)** – For content formatting
* **Telegram Bot API** – For message delivery
* **RSS Feeds** – For real-time data

---

## 🧠 How It Works

1. RSS feed fetches latest scholarship data
2. AI processes and formats the content
3. Filter removes unnecessary/repeated entries
4. Final message is sent to Telegram channel

---

## 📸 Workflow Architecture

RSS → AI Processing → Telegram
